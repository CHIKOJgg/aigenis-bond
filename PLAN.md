# План: Продающий SEO-оптимизированный лендинг Aigenis Bonds

## Проблема текущего лендинга
- Герой-секция говорит "Анализируйте облигации как профессиональный трейдер" — это про ФУНКЦИЮ, а не про РЕЗУЛЬТАТ
- Нет понимания, какую конкретно ДЕНЕЖНУЮ задачу решает продукт
- Нет SEO-тегов (title, description, Open Graph, structured data)
- Нет pain points / проблемного введения — сразу прыгает в фичи
- Нет social proof и trust signals
- Нет Urgency элементов
- Слабый CTA — "Начать бесплатный пробный период" вместо конкретной выгоды

## Стратегия
Показать пользователю: "Ты теряешь деньги, потому что не видишь лучшие облигации. Мы находим их за тебя."

### Ключевые сообщения
1. **Проблема**: На рынке 100+ облигаций. Как найти те, что дадут максимальную доходность при минимальном риске?
2. **Решение**: AI-скоринг находит лучшие облигации за секунды — те, что другие инвесторы пропускают
3. **Результат**: Вы получаете готовые к покупке облигации с оценкой риска/доходности

## Файлы для изменения

### 1. `frontend/index.html` — SEO meta теги
- Title: "Aigenis Bonds — AI-скоринг облигаций: найдите лучшие облигации для инвестиций"
- Meta description (160 символов)
- Open Graph теги (og:title, og:description, og:image, og:url)
- Twitter Card теги
- Canonical URL
- JSON-LD structured data (SoftwareApplication + FAQ)
- Preconnect к шрифтам

### 2. `frontend/src/i18n.tsx` — Новые переводы для продающих секций
Добавить ~30 новых ключей:
- Pain points секция (3 проблемы пользователя)
- ROI/ценность секция (конкретные цифры выгоды)
- Social proof (что говорят пользователи / метрики доверия)
- Urgency секция
- Improved hero copy (результат, а не функция)
- Improved CTA copy

### 3. `frontend/src/LandingPage.tsx` — Полная переработка структуры

**Новая структура лендинга:**

```
1. Navigation (без изменений, но с SEO-friendly nav links)

2. Hero Section (ПЕРЕРАБОТАН)
   - Badge: "100+ облигаций проанализированы за секунды"
   - H1: "Найдите облигации, которые дадут максимальный доход"
   - Sub: "AI-скоринг анализирует рынок и показывает лучшие机会 для инвестиций"
   - CTA: "Посмотреть лучшие облигации бесплатно"
   - Подпись: "7 дней Pro-доступа. Без привязки карты."

3. Live Widget ( BestBondsWidget ) — СОХРАНЯЕМ, это лучший sales-элемент
   - Показывает РЕАЛЬНЫЕ данные — лучший proof

4. Pain Points / Проблемы (НОВАЯ СЕКЦИЯ)
   - "Устали искать лучшие облигации вручную?"
   - "Не уверены, какой риск у вашей инвестиции?"
   - "Пропускаете выгодные предложения из-за нехватки времени?"
   → Каждая проблема решается конкретной фичей продукта

5. Stats Bar (ПЕРЕРАБОТАН)
   - Больше фокус на доверие: "100+ облигаций проанализировано", "6 валют", "24/7 мониторинг"
   - Добавить: "Работает с данными MOEX"

6. Features (ПЕРЕРАБОТАНы)
   - Каждая фича = "Какую задачу решает" + "Какой результат получаете"
   - Переименовать заголовки с фокусом на результат:
     - "Автоматический сборщик" → "Никаких упущенных данных"
     - "Скоринг риска/доходности" → "Знайте риск каждой облигации"
     - и т.д.

7. How It Works (УПРОЩЁН)
   - 3 шага: Смотрите → Выбирайте → Зарабатывайте

8. ROI / Ценность (НОВАЯ СЕКЦИЯ)
   - "Один найденный bond с YTM +2% = окупаемость подписки на год"
   - Конкретные цифры

9. Pricing (ПЕРЕРАБОТАН)
   - Привязать к конкретной выгоде, а не к фичам

10. CTA Section (ПЕРЕРАБОТАН)
    - Urgency: "Рынок не ждёт"
    - Конкретный результат: "Узнайте, какие облигации покупать сегодня"

11. Footer
```
