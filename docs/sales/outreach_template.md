# B2B-outreach шаблон (партнёрский white-label Aigenis Bonds)

Используется для холодного исходящего (outbound) на ЛПР белорусских/российских
брокеров, банков, финтех- и EdTech-платформ. Цель — подключить white-label
виджет облигаций / embedded-инвестиции и выдать партнёрский API-ключ самообслуживанием.

Материалы для прикрепления/ссылок:
- One-pager: docs/sales/one_pager.md (публичная B2B-лендинг-страница: /partners)
- Teaser: docs/sales/teaser.md
- Демо: https://web.aigenis.by  (кнопка "Купить" на карточке облигации)
- Self-serve онбординг: https://aigenis.by/partners  → заявка → API-ключ + реф-код выпускаются автоматически

## Ценность для партнёра (1 абзац)
«Добавьте своим клиентам покупку облигаций Минфина и корпоративных выпусков
прямо в ваше приложение/сайт — без лицензии и разработки. Aigenis даёт
white-label виджет, API и реферальную программу (комиссия с оборота).
Подключение за час, первый ключ выпускается автоматически по заявке.»

## Telegram DM (первое касание)
```
Здравствуйте, {Имя}! Я из Aigenis — мы даём брокерскую инфраструктуру
облигаций как white-label: ваш бренд, наше ядро (покупка ВГДО/корпоблигаций,
KYC, депозитарий). Хотим предложить встроить облигации в {продукт партнёра}
за час и делить комиссию с оборота. Могу показать демо и сразу выдать
тестовый API-ключ — удобно глянуть?
```

## Второе касание (через 3–4 дня, если нет ответа)
```

для {Компания} можно получить самостоятельно за 2 минуты на
https://aigenis.by/partners. Если интересно — покажу, как виджет
выглядит в реальном приложении.
```

## Возражения → ответы
- «У нас уже есть брокерская лицензия» → «Отлично — наше API избавит от
  разработки UI/парсинга котировок; вы остаётесь с клиентом, мы — ядро.»
- «Нет времени на интеграцию» → «Виджет встаёт за час, без кода.
  Или дайте нам доступ — вставим сами.»
- «Юр. вопросы/комплаенс» → «Работаем в рамках БВФБ и бел. регулирования;
  готовы предоставить документы и договор партнёра.»
