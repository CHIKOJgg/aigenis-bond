---

**Aigenis Bonds — аналитика облигаций «под ключ»**

Готовый продукт (не MVP), который собирает живые рыночные данные
(RUB корпоративные + USD/EUR евробонды через публичный MOEX ISS; BYN —
через партнёрский источник), оценивает облигации по скорингу 0–100,
строит ML-рекомендации (buy/hold/wait/avoid) и предоставляет инструменты
профессионального Fixed Income Desk: кривые доходности, duration,
relative value, carry, РЕПО и стресс-тесты.

**Для кого:**
- Брокеры / агрегаторы котировок — white-label аналитика на вашем бренде.
- Финтех-приложения — встройте Bond API + виджет «Топ облигаций».
- EdTech по инвестициям — готовая учебная платформа с реальными данными.

**Почему сейчас:**
- Production-ready: 102 теста, CI, биллинг (ЮKassa + Stars), партнёрский API.
- Есть публичный DEMO-режим — покажем продукт за 1 клик.
- Гибкая сделка: лицензия / white-label / покупка компании.
