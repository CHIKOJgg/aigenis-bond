# Aigenis Bonds — CMO-аудит (жёсткий)

> Цель: честный маркетинговый диагноз и приоритеты роста. Дополняет
> `gtm_launch.md`, `one_pager.md`, `teaser.md`. Дата: 2026-07.

## 1. Диагноз позиционирования (главная проблема)

- **Две разные аудитории в одном продукте.** B2C-розница хочет «стоит ли
  купить» (Score), а Desk-фичи (duration, NS-кривая, repo, RV, stress) —
  для профессионалов. Тиры **29 / 99 BYN/мес** при этом смешны для профи
  (они бы платили $150–300/мес за Desk) и дороги для частника, которому
  брокер (Тинькофф/Альфа/Беларусбанк) отдаёт YTM бесплатно. Продукт
  «ни для кого конкретно».
- **Рассинхрон в материалах = потеря доверия у buyer.** В `teaser.md` до
  сих пор «облигациям 6 валют», хотя в `one_pager.md` уже честно поправили
  на RUB+USD/EUR (MOEX) + BYN (платный). Расхождение one_pager ↔ teaser —
  красный флаг на due diligence при M&A.

## 2. Что добавить (product/marketing)

- **Публичные SEO-страницы по каждой облигации** (без логина): доходность,
  график, Score, вердикт. Главный бесплатный двигатель трафика и витрина
  ценности.
- **Лидерборды / «Топ-10 облигаций РФ по доходности сегодня»** — link-bait
  + щит для партнёров и органики.
- **Telegram Mini App вместо отдельного веб-сайта** — один сёрфейс, меньше
  трения, проще воронка (см. §7).
- **Налоговый/чистый ROI-калькулятор** (после НДФЛ/подоходного) — то, что
  реально волнует частника, а не DV01.
- **Гайд-хаб** («что такое duration простыми словами», «carry для чайников»)
  — связка с EdTech-B2B и органический трафик.

## 3. Расширение доступа

- `DEMO_MODE=1` уже есть — вынести в постоянный публичный `demo.`-инстанс
  как витрину + SEO-базу (см. `DEPLOYMENT.md`, раздел 6).
- Mini App + webhook + Cloudflare Tunnel (без белого IP) — доступ «за 0».
- Виджет партнёрам (`/widget`, `WidgetPage.tsx`) масштабировать как канал
  дистрибуции.

## 4. Продажа БЕЗ вложений

- Affiliate на 5–10 каналов/блогов по инвестициям (реализовано: ключи +
  ref-code + % с подписок).
- Контент-серия в TG-канале: «разбор облигации по Score + стресс-тест»,
  2 поста/нед.
- SEO-страницы (§2) — долгосрочный органик без бюджета.
- **Критично:** сделать `DATA_SOURCE=moex` дефолтным, чтобы продукт жил
  без платного BYN-логина (снять операционный риск из `one_pager.md`).

## 5. Продажа С вложениями

- Купить качественные фиды (полнота/свежесть BYN и мультивалюта) — убрать
  зависимость от парсинга-скрапинга.
- Контент-команда + платный трафик (Telegram Ads, ВК таргет на аудиторию
  «инвестиции»).
- Мобильное приложение / PWA, выход на EU/US рынки, SDR для B2B-outbound.

## 6. SEO (конкретно)

- Статические страницы облигаций + лидерборды + гайды.
- `Schema.org` финансовая разметка, `sitemap.xml`, индексация через
  регулярный пересбор.
- Long-tail: «доходность облигации Газпром 2027», «купить евробонд USD что
  выгоднее», «топ корпоративных облигаций по доходности».

## 7. Упрощение пайплайна «увидел → попробовал → купил»

Сейчас: заход → 7 дней Pro авто → paywall 402 → выбор Stars/YooKassa →
оплата. Трение — разделение бот/сайт и два платёжных рельса.

- Один CTA «Подписаться» → **инлайн-оплата одним тапом в Mini App/боте**
  (Stars уже идемпотентен по `telegram_payment_charge_id` — работает $0).
- Убрать отдельный сайт-пэйвол; сделать Mini App единой точкой покупки.
- Триал без карты (уже) + напоминания (`reminders_daily`, уже) — оставить.

## 8. B2B (реальный клин)

- Продавать не B2C-подписку, а **white-label виджет/API + revenue-share**
  брокерам, робо-эдвайзерам, финмедиа (smart-lab, banki.ru), EdTech.
- Outbound: список 30–50 ЛПР → демо через `demo.`-инстанс →
  `one_pager`/`teaser`.
- **Сделать отдельную B2B-цену** (сейчас в документах только M&A-фрейминг,
  affiliate-комиссия `REFERRAL_COMMISSION_PCT` по умолчанию 0).
