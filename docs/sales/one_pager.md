# Aigenis Bonds — One-Pager (для M&A / лицензии / white-label)

**Коротко.** Aigenis Bonds — production-ready SaaS для анализа облигаций
фиксированного дохода (Fixed Income). Полный цикл: сбор данных → скоринг →
ML → Desk-аналитика → продажа подписки (B2C + B2B).

## Что внутри (готовый актив)
- **Парсер** ежедневно собирает 100+ облигаций с котировками, купонами,
  доходностями и историей.
- **Скоринг 0–100** (risk/reward) + буквенные тиры.
- **ML** (scikit-learn): регрессия YTM, классификатор
  buy/hold/wait/avoid, объяснимые рекомендации.
- **Fixed Income Desk**: кривая доходности (Nelson-Siegel), duration,
  relative value, carry, РЕПО, стресс-тесты.
- **Telegram-бот** (команды + Stars-платежи) и **Web+API** (React + FastAPI).
- **B2B Partner API**: ключи, webhooks, read-only аналитика + атрибуция
  рефералов/партнёров.
- **Публичная B2B-витрина** `/partners` — white-label/лицензия/affiliate
  одним экраном + CTA в Telegram.
- **Публичные SEO-страницы** `/bonds` и `/bonds/{internal_id}` —
  органический трафик без бюджета (индексируются краулерами).
- **Биллинг**: ЮKassa (карты/SBP) + Telegram Stars, paywall 402.
- **Observability**: Prometheus-метрики, 102 теста, CI, Alembic-миграции.

## Охват рынков (честно)
- **Живые данные без платного логина:** RUB (корпоративные, MOEX TQCB) +
  USD/EUR евробонды (MOEX TQOB) через публичный fallback-источник.
- **BYN:** через партнёрский платный источник (основной сбор).
- **Расширение** на EU/US рынки — в roadmap (требует подключения новых фидов).

## Метрики (на момент аудита)
- 100+ облигаций, 24/7 мониторинг.
- 102 авто-теста (V1–V5), покрытие парсер/скоринг/Desk/ML/API.
- Готовый публичный DEMO-режим (read-only, watermark) для презентации buyers.

## Операционный риск (важно для buyer)
Парсер основного (BYN) источника требует платного логина. Зависимость
частично снята публичным fallback (MOEX RUB + USD/EUR евробонды); полное
снятие и расширение на новые рынки — в roadmap (подключение фидов).
